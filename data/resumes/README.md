# Sample resumes

Everything in this folder except the three files below is `.gitignore`d —
drop real resumes here for local testing without risk of committing candidate
PII.

The exceptions are **synthetic, fictional demo resumes** (fake names, fake
emails, fake employers) checked in on purpose so that `make run`, the CLI
Quick Start commands, and the README's sample output are all reproducible
from a fresh clone with zero setup:

- `sample.pdf` — "Alex Rivera", strong fit for `data/job_descriptions/sample_jd.txt`
- `sample_mid_level.txt` — "Priya Nair", partial fit
- `sample_junior_frontend.txt` — "Sam Okafor", weak fit

None of these represent real people.
